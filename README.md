# Sundrela Krishi Kendra

A modern web application for **Sundrela Krishi Kendra**, an agricultural equipment service provider based in Village Kaluheda, District Ujjain, Madhya Pradesh.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (with SWC)
- **Tailwind CSS** + **shadcn/ui** (sonner only)
- **Framer Motion** for animations
- **React Router DOM** for routing
- **Sonner** for toast notifications

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
